# 🔳 QR Code Generator

A simple Python application that generates QR codes from user-provided text or URLs.

## 🚀 Features

- Generate QR codes instantly
- Supports text and URLs
- Saves QR codes as PNG images
- Lightweight and beginner-friendly

## 🛠️ Technologies Used

- Python
- qrcode
- Pillow

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/Pranshu-code123/qr-code-generator.git
```

2. Install dependencies

```bash
pip install qrcode pillow
```

3. Run the application

```bash
python main.py
```
